# Anime Tracking Project

## Overview
This project is designed for tracking anime series and episodes. Users can manage their watch lists, mark episodes as watched, and receive updates on new episodes.

## Main Features
- Track watched episodes and series
- Create and manage a watchlist
- Notifications for new episodes
- User-friendly interface

## Technologies Used
- **Frontend**: React.js
- **Backend**: Node.js with Express
- **Database**: MongoDB
- **Deployment**: Heroku

## Installation Prerequisites
1. Ensure you have Node.js installed.
2. Clone the repository:
   ```bash
   git clone https://github.com/Luslac/Anime-Tracking.git
   cd Anime-Tracking
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Set up your database and configure environment variables as needed.

## Usage Examples
1. Start the development server:
   ```bash
   npm run dev
   ```
2. Visit `http://localhost:3000` to view the app.
3. Follow the prompts to add anime series to your watchlist.
